空间与表层参考卡

让页面显贵的杠杆，附带它们的数值。来自第 11 课。内容相同、字体相同、颜色相同：改变这些，页面的 「身价」就变了。

杠杆 · 该伸手去拿什么

  • 宏观留白。把你的标准内边距加倍。安静、高级的区块用 py-24 到 py-40（最宽敞时可到 py-48）。空间是让页面显贵最便宜的办法，也是默认值 最先出错的地方。
  • 行宽 = 65ch。正文文字每行控制在大约 65 个字符（max-width: 65ch）。再宽就会 读起来像一份没有排版的文档。
  • 圆角锁定。选定一套圆角刻度，然后在所有地方都守住它（全直角、全柔和 12–16px，或全药丸形用于交互元素）。混用圆角只有在有明文规则、且处处遵守时才被允许。
  • 双层描边（同心圆角）。用一个带内边距的外壳把内容核心包起来。内层圆角等于外层圆角减去 内边距，于是曲线彼此嵌套：外 20px − 内边距 8px = 内 12px。
  • 带色调的弥散阴影。大幅模糊，投射得远，负扩散，染上背景的色调，外加一条 1px 的内部高光。 绝不用又小又黑的 shadow-md 投影。
  • 颗粒，约 3–6%。一层微弱的胶片颗粒会读起来像实体的纸张。让它待在一个 fixed; pointer-events:none 的图层上，绝不放在滚动容器里。

配方 · 照抄这些数值

/* double-bezel card */
.shell { padding: 8px; border-radius: 20px; background: rgba(28,26,22,.035); border: 1px solid rgba(28,26,22,.07); }
.core  { border-radius: 12px; /* 20 − 8 */ background: #fff; }

/* diffusion shadow, tinted to ink */
box-shadow: inset 0 1px 0 rgba(255,255,255,.6),
           0 30px 60px -28px rgba(28,26,22,.30);

/* banned: the cheap default */
box-shadow: 0 4px 6px rgba(0,0,0,.3); /* shadow-md */

廉价痕迹 · 该去掉什么

  • 紧绷、统一的内边距。到处都是默认组件间距。修法：把区块内边距加倍，让宏观间距区别于微观 间距。
  • 刺眼的黑色阴影。shadow-md 或 rgba(0,0,0,.3)。修法：换成柔和、 投射得远、带色调的弥散阴影。
  • 混用的圆角。一张 4px 的卡片配一个 16px 的按钮。修法：锁定一套圆角系统。
  • 每个元素都套边框。每个元素周围都有一圈 1px 灰框。修法：改用发丝线、留白和阴影来做分隔。
  • 铺满全宽的正文。文字行贯穿整个容器宽度。修法：把行宽控制在 65ch 以内。
