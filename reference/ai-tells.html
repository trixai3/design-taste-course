The AI-Tells Cheat Sheet

The concrete signatures that make an interface read as generated. This sheet grows across the course; the fifteen below come from Lesson 01. Source shorthand: DTF = design-taste-frontend/SKILL.md.

Visual & CSS

glow

AI purple/blue gradient

Violet-to-blue mesh + neon button glow. The single most recognizable AI fingerprint. Fix: warm neutral base, one flat accent, no glow.

DTF §4.2 (the Lila rule) · §9.A
#000000

Pure black

#000 backgrounds or text kill depth and read as digital-default. Fix: off-black (zinc-950 / warm charcoal).

DTF §9.A · §8.B

Typography

Aa

Inter everywhere

The character-neutral default. Not ugly, just invisible, so every page feels templated. Fix: a display face with a voice (Cabinet Grotesk, Satoshi…).

DTF §4.1 · §9.B
—

The em-dash flourish

The model's signature punctuation crutch. Banned outright: zero anywhere on the page. Fix: period, comma, colon, or parentheses.

DTF §9.G

Layout

centered
H1

Centered hero over dark mesh

The default hero the model always tries first. Signals no composition was chosen. Fix: asymmetric split, text one side, asset the other.

DTF §0.D · §4.3

Three equal feature cards

Icon + bold word + one sentence, three across, bordered and shadowed. Fix: asymmetric list, zigzag, or varied bento with rhythm.

DTF §9.C · §4.7
00 / INDEX

Section-number eyebrow

Numbered labels over every header ('00 / How it works'). Templated rhythm. Fix: max 1 eyebrow per 3 sections; name it in plain words.

DTF §4.7 · §9.F
$ run
✓ ok
✓ merge

Div-based fake screenshot

Fake terminal / dashboard / traffic-light window built from styled divs. The #1 Tell. Fix: a real screenshot, generated image, or none.

DTF §4.8 · §9.E

Content & Copy

Nexus

Startup-slop brand name

Nexus, Acme, Apex, Cloudly: generic invented names, often with a token-colored letter. Fix: a plain, sayable, descriptive name.

DTF §9.D
Unleash

Filler-verb copy

Elevate, Unleash, Seamless, Next-Gen: clichés that say nothing. Fix: the concrete action ('Start a review').

DTF §9.D · §4.9
99.99%

Fake-perfect statistic

Suspiciously round invented metrics with no source read as hallucinated. Fix: real, messy, sourced numbers, or none.

DTF §9.D · §4.9
Sarah Chen
Developer

The Jane Doe testimonial

Generic name + vague role + no company. Attribution with zero specificity. Fix: real name, specific role, real company.

DTF §9.D
✦ v2.0 BETA

Version label in the hero

'EARLY ACCESS', 'INVITE-ONLY PREVIEW' as an eyebrow. Status theater, not value. Fix: an eyebrow that states what the product does.

DTF §9.F
↓ Scroll

Scroll cue

'↓ Scroll to explore.' The user is on the hero; they know how to scroll. Fix: delete it. Let content pull the eye down.

DTF §9.F
● Live

Decorative status dots

Colored ● before nav items or a fake 'Live' badge, used as decoration not real state. Fix: drop it, or make the dot mean something.
