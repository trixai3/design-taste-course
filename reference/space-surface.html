The Space & Surface Sheet

The levers that make a page feel expensive, with their numbers. From Lesson 11. Same content, same font, same color: change these and the page changes price.

Levers · what to reach for

  • Macro-whitespace. Double your standard padding. Calm, premium sections run py-24 to py-40 (up to py-48 at the most spacious). Space is the cheapest way to look expensive, and the first thing defaults get wrong.
  • Measure = 65ch. Cap body text near 65 characters per line (max-width: 65ch). Wider reads as an unstyled document.
  • Radius lock. Pick ONE corner-radius scale and hold it everywhere (all-sharp, all-soft 12–16px, or all-pill for interactive). Mixed radii are allowed only under a documented rule followed everywhere.
  • Double-bezel (concentric radius). Wrap a content core in an outer shell with padding. The inner radius is the outer radius minus the padding, so the curves nest: outer 20px − pad 8px = inner 12px.
  • Diffuse tinted shadow. Large blur, cast far, negative spread, tinted to the background hue, plus a 1px inner highlight. Never the small black shadow-md drop.
  • Grain, ~3–6%. A faint film-grain reads as physical paper. Keep it on a fixed; pointer-events:none layer, never on a scrolling container.

Recipes · copy the numbers

/* double-bezel card */
.shell { padding: 8px; border-radius: 20px; background: rgba(28,26,22,.035); border: 1px solid rgba(28,26,22,.07); }
.core  { border-radius: 12px; /* 20 − 8 */ background: #fff; }

/* diffusion shadow, tinted to ink */
box-shadow: inset 0 1px 0 rgba(255,255,255,.6),
           0 30px 60px -28px rgba(28,26,22,.30);

/* banned: the cheap default */
box-shadow: 0 4px 6px rgba(0,0,0,.3); /* shadow-md */

Cheap tells · what to remove

  • Tight, uniform padding. Default component spacing everywhere. Fix: double the section padding, vary macro from micro.
  • Harsh black shadow. shadow-md or rgba(0,0,0,.3). Fix: a soft, far, tinted diffusion shadow.
  • Mixed radii. A 4px card with a 16px button. Fix: lock one radius system.
  • Borders on everything. A 1px gray box around every element. Fix: hairlines, whitespace, and shadow do the separating.
  • Full-width body text. Lines running the whole container. Fix: cap at 65ch.
